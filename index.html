Udactiy Front-End Project: neighborhood map

This project is part of Udacity Front-End Web Developer Nanodegree. It's intended to be a single page application featuring a map which contains a list of locations (at least 5) you would like to show. The app will show selected locations in Shanghai with Google Maps API, and get wikipedia link via jQuery ajax (if the location has the wikipedia link). Also, you can filter the location using input in siderbar. There are pinned locations in the map which users can click on and infomation will be showed in the siderbar instead of list.
